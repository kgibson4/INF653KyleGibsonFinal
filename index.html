US States REST API

Welcome to the US States Data API. This service provides comprehensive information about all 50 United States, including capitals, nicknames, populations, and fun facts.

Core Endpoints:

  • /states/ - Get all state data
  • /states/:state - Get data for a specific state (use abbreviations like KS, NY)
  • /states/:state/funfact - Get a random fun fact

This API is built using Node.js, Express, and MongoDB.
